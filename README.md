# 📊 Customer Behavior & Sales Performance Dashboard

## 📖 Overview
This project presents an interactive dashboard built using Power BI to analyze customer behavior and overall sales performance.

The goal is to extract meaningful insights from retail sales data, understand customer purchasing patterns, and support better business decision-making.

---

## 📊 Key Features
- KPI Metrics:
  - Total Revenue
  - Total Orders
  - Total Quantity
  - Average Order Value
- Sales analysis by product category
- Customer segmentation by age group
- Monthly sales trend analysis
- Customer purchasing behavior insights

---

## 📈 Key Insights
- Middle-aged customers generate the highest revenue contribution
- Sales are relatively balanced across product categories
- Average order value is $456, indicating moderate customer spending behavior
- Sales trends show fluctuations with noticeable peak periods
- Customers purchase an average of ~2.5 items per order

---

## 🛠 Tools Used
- Power BI
- Excel
- DAX (Data Analysis Expressions)

---

## 📂 Dataset
The dataset contains retail transaction data, including:
- Sales revenue
- Quantity sold
- Customer age groups
- Product categories
- Order transactions

---

## 🚀 How to Use
1. Download the `.pbix` file from this repository  
2. Open it using Power BI Desktop  
3. Explore the interactive dashboard and insights
